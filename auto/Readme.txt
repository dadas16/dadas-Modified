Features:
==========
a) The auto complete script pre-populates the search term in a textbox when a letter or word is typed.
b) Auto search provides the functionality for search text suggestions.
c) Allows users to select items from the predefined list.

Installation:
==============

File permissions:
a) Unzip  autocomplete-search.zip.
b) Set read, write permission for the file autocomplete-search/config.php for Linux
users.

Creating a database:
a) Create database for JQuery Autocomplete Search Script, using any name.
b) Run the file autocomplete-search/install.php in your browser.
c) Provide the database details in the given fields.
   Note: Enter your tablename & fieldname which you want to list in the autocomplete search.
d) Click on "Install" and you will get "sucessfully installed" message.
e) After installing, execute the file autocomplete-search/index.php in your browser.
